Synopsis :

The aim of this study is to accurately be able to quanitfy correctness of an activity. The used in this study is from the UCI-HAR library and is focused on dumbell lifts. link

We will do the following in this study :

  1. Brief EDA of the data to understand what the structure is like

  2. Clean the data

  3. Train ML classifiers and select the best ones based metrics

  4. Predict activities based on the data in the testing set.

Important part of this computation is parallel processing, the function used is defined here. It outputs model run time that allows for us to make model selection decisions later on.

# Parallel processing function,to help reduce runtime
runparproc<-function(model.fun=x()){
  require(parallel)
  require(doParallel)
  # convention to leave 1 core for OS
  cluster <- makeCluster(detectCores() - 1) 
  registerDoParallel(cluster)
  ptm <- proc.time()
  model.fun
  elap <- proc.time() - ptm
  stopCluster(cluster)
  print(elap)
}
